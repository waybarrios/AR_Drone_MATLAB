# AR_Drone_MATLAB


###How To Use:
Connect to AR.Drone via wi-fi
navigate to command window
type

'drone = ARDrone'
'drone.control'

follow on-screen instructions

Tested on mac osx and windows
uses AR.Drone firmware v1.7

Allows user to:
control drone using keyboard
view live navdata feed


##Other requirements 	
* AR.Drone v1.7
* MATLAB release 	
* MATLAB 7.5 (R2007b)
